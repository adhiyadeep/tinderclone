    ### Running the project

Clone this repository :

```
git clone https://github.com/adhiyadeep/tinderclone.git
cd tinderclone
```

Install packages :

```
npm install
```

When installation is complete, run with version of your choice :

```bash
react-native run-ios
# or
react-native run-android
